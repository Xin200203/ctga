# 阶段实验与指标

## 主指标

- `AP / AP50 / AP25`
- `fragmentation`
- `merge error`
- `ID switches`
- `latency / FPS`

## 阶段 Gate

### Gate 1: 只看 Layer-1

目标：

- fragmentation 明显下降
- merge error 不显著上升

如果不成立：

- 先不要推进 Layer-2
- 优先检查 primitive 质量、signed graph 组装、历史 veto 逻辑

### Gate 2: 固定 Layer-1，对比 Hungarian

目标：

- 在相同 current objects 上
- `beam_qap` 比 `hungarian` 明显降低 ID switches

如果不成立：

- 优先检查 relation feature 设计
- 不要先扩大模型规模

### Gate 3: 学边 vs 手工边

目标：

- 在相同 solver 下，learned scorer 明显优于 hand-crafted 版本

如果不成立：

- 回退到更简单的 edge MLP
- 重新检查标签质量和 hard-negative 构造

## Baseline

- ESAM / ESAM-E
- AutoSeg3D
- Unary Hungarian baseline
- OnlineAnySeg 作为空间候选与 visibility-aware containment 参考

## 核心消融

- `graph1_solver = threshold_cc` vs `gasp`
- `solver_mode = hungarian` vs `beam_qap`
- `use_edge_mt = false`
- `use_edge_pt = false`
- `use_visible_containment = false`
- `use_pairwise_relation = false`
