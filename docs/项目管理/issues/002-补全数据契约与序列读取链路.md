# [M1] 补全数据契约与序列读取链路

## 背景

目前 `FramePacket`、`PosedRGBDSequence` 和 mask cache 已经有基础实现，但还需要在真实数据集场景下进一步固化约定。

## 目标

- 对齐 ScanNet / ScanNet200 / SceneNN 的目录约定
- 明确缓存命名规则和字段规范
- 补足 loader 侧的边界情况处理

## 输出

- 更稳定的数据读取说明
- 必要的数据样例与校验脚本
- 数据约定文档回填

## 验收标准

- 文档中的数据契约与代码实现保持一致
- 在远程环境中可用真实样本完成一次最小读取检查
