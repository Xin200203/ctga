# [M1] 建立远程运行与评测基线

## 背景

仓库已经具备基础模块和 smoke tests，接下来需要在远程服务器或工作站上建立正式的运行、回放和评测基线。

## 目标

- 固化远程环境安装流程
- 跑通 `pytest`、`replay_sequence.py` 和最小 online inference
- 明确 baseline 对比的第一轮命令入口

## 输出

- 远程运行说明
- baseline 运行记录
- evaluator 输出格式说明

## 验收标准

- 远程环境中能通过 smoke tests
- 能完成至少一次 sequence replay
- 能导出 debug packet 并读取

## 建议 Labels

- `evaluation`
- `implementation`
- `priority:medium`
