# 阶段路线图

## 阶段 0

目标：

- 跑通 ESAM 的在线接口与 custom dataset 输入
- 跑通 OnlineAnySeg 的空间关联主链路
- 确认 GASP 的 signed graph partition 接口可调用

## 阶段 1

目标：

- 只实现 Layer-1 evidence graph
- 验证同帧图是否能降低 fragmentation

交付：

- primitive builder
- evidence graph builder
- signed graph assembler
- GASP solver wrapper
- current object builder

## 阶段 2

目标：

- 固定 Layer-1 输出
- 只做 unary + Hungarian 的 temporal baseline

价值：

- 建立自己的强一阶基线
- 为后续 graph matching 提供干净对照

## 阶段 3

目标：

- 在 active-set 上引入 candidate-gated 二阶 graph matching
- 对重复物体场景重点看 ID switches 与 association accuracy

## 阶段 4

目标：

- 用 learned edge scorer 替代手工边分数
- 决定是否值得继续 joint finetune

## 阶段 5

目标：

- 后置接入 open-vocabulary semantics
- 保持语义层与主关联环节解耦
